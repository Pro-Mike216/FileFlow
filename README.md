# FileFlow

A Python automation tool that organizes files into folders based on their file extensions.

## Features

- Organizes files automatically
- Supports multiple file categories:
  - Pictures
  - Videos
  - Documents
  - Code
  - Audio
  - Archives

## Technologies

- Python
- pathlib
- os
- shutil

## How it works

The program scans a folder, identifies files based on their extensions, and moves them into appropriate categories.

## Usage

Run the program:

python organizer.py

Then enter the folder path you want to organize.
